JDAT Documentation

JDAT is a data storage format.
It is NOT a programming language.

1. File structure

A JDAT file contains entries.
Each entry is composed of:

  • Metadata between ( )
  • Data between { }

2. Metadata fields

  • n : name
  • l : link (identifier)
  • t : type

Types:

  • t:1 → structured data
  • t:2 → raw code / blob (never executed)

3. Example (t:1)

(n:bob l:bob t:1 {
    name: Bob
    email: bob@example.com
    password: 12345ABCDE._.
})

4. Example (t:2)

(n:python_code l:python_example t:2 {
from flask import Flask
app = Flask(__name__)
})

5. JDAT Shell

  • open file.jdat
  • list / list /all
  • crypt <link> <key>
  • decrypt <link> <keyfile>
  • goto <path>

6. Philosophy

  • Simple
  • Portable
  • Human-readable
  • Tool-agnostic
